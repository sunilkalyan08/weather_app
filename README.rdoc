== README

Versions to be check

1. Ruby 2.4.1
2. Rails 5.1.1

Steps to run the application

1. Clone the repo via SSH or https.
2. Goto your project path (i.e. ~/weather-app) in Finder/File Explorer.
3. Then open `**Terminal**` and goto path `~/weather-app`
4. Type `bundle install` (assuming you have bundler installed) to install the necessary gems for the project.
